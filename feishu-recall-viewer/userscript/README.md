# 本地不入侵方案（油猴脚本）

适合：**不想接飞书开放平台、不想申请企业敏感权限、只在自己电脑上看自己已经看过的消息**。

## 和「机器人方案」的区别

| | 油猴本地归档（本方案） | 开放平台机器人（`npm start`） |
| --- | --- | --- |
| 是否要企业管理员审批 | 否 | 群全量消息通常要敏感权限审批 |
| 是否改飞书客户端/读加密库 | 否 | 否 |
| 覆盖范围 | **仅本浏览器里你已经打开并看到过的消息** | 机器人所在会话、权限范围内的消息 |
| 脚本启动前的历史撤回 | **无法还原** | 若当时机器人已在群且收到过，可以 |
| 数据存哪 | 本机浏览器 IndexedDB | 你自己的服务器/本机 JSON |

## 原理（不入侵）

1. 你在飞书**网页版**正常看聊天时，脚本把屏幕上已经渲染出来的消息文本存进本机 IndexedDB。  
2. 对方撤回后，页面变成「消息已撤回」；脚本用之前存的文本在气泡下方显示「本地归档 · 撤回前原文」，并在右下角面板列出。  
3. **不**注入飞书私有 API、**不**读客户端本地数据库、**不**绕过权限。本质是「你看过的内容的本地笔记」。

## 安装

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/)（Chrome / Edge / Firefox）  
2. 打开脚本文件：[`feishu-recall-local.user.js`](./feishu-recall-local.user.js)  
3. Tampermonkey →「添加新脚本」→ 粘贴全部内容 → 保存  
4. 打开飞书网页版（`*.feishu.cn` / `*.larksuite.com` 等），右下角出现 **「撤回归档」** 按钮即表示生效  

## 使用注意

- 先让会话消息在屏幕上出现过，再撤回，才能还原。  
- 换浏览器 / 清站点数据 / 无痕模式，本地归档会丢。  
- 飞书改版 DOM 后，识别可能变弱；若失效，把页面结构反馈后可再适配。  
- 仅用于你有权查看的会话，遵守公司规定；不要用于窥探他人隐私。

## 做不到的事（请预期）

- 还原你**从没打开过**的会话里的撤回  
- 还原脚本安装**之前**就已经撤回的消息  
- 像「破解版」那样读取飞书客户端本地加密缓存  

若需要群内自动、全量、可服务端检索，请用同目录下的开放平台机器人方案（见上级 [README.md](../README.md)）。
