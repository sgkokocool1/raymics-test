接入说明（摘要）
1. 飞书开放平台创建企业自建应用，开启机器人能力
2. 权限：至少开通 im:message / im:message:readonly；若要收群内全部消息还需 im:message.group_msg（敏感权限）
3. 事件订阅：请求地址填 https://你的域名/webhook/event，订阅 im.message.receive_v1 与 im.message.recalled_v1
4. 把机器人拉进需要监控的群；撤回事件本身不含原文，必须先收到并保存消息
5. 详细步骤见项目 README
