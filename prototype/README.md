# 具身智能数据平台 · 可运行原型

一个纯前端、开箱即用的交互原型，演示[设计文档](../docs/具身智能数据平台设计调研.md)中的全部核心页面与数据全生命周期流转。

## 运行方式

方式一（推荐）：本地静态服务器

```bash
cd prototype
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

方式二：直接双击 `index.html` 用浏览器打开（`file://` 亦可运行，脚本均为非 module）。

> 图表库（ECharts）与流程图库（Mermaid）通过 CDN 加载，首次打开需联网。

## 技术栈

- 原生 HTML/CSS/JS（无需构建），单页应用（SPA）
- [ECharts](https://echarts.apache.org/) 统计图表
- [Mermaid](https://mermaid.js.org/) 架构与流程图
- 全部为内置 mock 数据（`data.js`，320 条模拟 episode）

## 页面一览

| 页面 | 对应设计环节 |
| --- | --- |
| 📊 总览看板 | 原始数据分布、分类占比、Tag 占比、执行漏斗、训练回流 |
| 🗂️ 数据浏览筛选 | 多维组合筛选 + 列表；点击进入详情 |
| 📄 数据详情 | 多路相机同步回放（占位）+ 传感器曲线 + 元数据 + 生命周期 |
| ⚙️ 清洗·预处理 | 清洗规则命中、质量评分、预处理步骤、任务执行情况 |
| ✏️ 标注工作台 | AI 预标注 + bbox/keypoint、时序分割、标注工作流 |
| 💎 数据资产 | 资产化、版本、标注、复用 |
| 🏷️ 标签管理 | 多维标签体系、Treemap 分布、覆盖率 |
| 🔎 检索中心 | 结构化 + 全文 + 语义混合检索、分面下钻、相似检索 |
| 📦 数据集 | 圈选/切分/多格式导出 |
| 🚀 训练任务 | 训练状态、数据飞轮指标回流 |
| 🧩 架构·流程图 | 分层架构 / 生命周期 / 状态机 / ER / 检索链路（Mermaid） |
